Al darme cuenta de que la V1 de mi sitio se estaba volviendo innecesariamente compleja, decidí empezar de cero creando la V2 de este sitio web que estás viendo actualmente.

Este sitio web está compuesto de **HTML5** y **CSS3** estáticos construidos con el Framework **Astro**. Esto hace que el sitio sea extremadamente ligero y rápido.

Algunas características de esta versión:

- Todo el contenido del sitio se almacena y gestiona en archivos **Markdown**.
- También fue una oportunidad para probar y aprender el framework Astro, muy adecuado para sitios centrados en el contenido.
- Además, migré de Node.js a **Bun** para la gestión del runtime JS durante el desarrollo.
