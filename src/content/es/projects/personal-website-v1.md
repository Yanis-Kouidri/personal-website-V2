Mi primer gran proyecto web, el predecesor de este sitio. El objetivo era aprender y practicar el desarrollo web full stack, así que elegí el stack **MERN** (MongoDB Express React Node.js).

Diseñé:

- El front-end en **React**
- El back-end en **Node.js** con Express
- El almacenamiento en una base de datos **MongoDB**

También añadí un sistema de carga de archivos con _multer_ y un mecanismo **CRUD**. ¡Durante este proyecto aprendí mucho!
