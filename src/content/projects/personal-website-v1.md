Mon premier gros projet web, c'est l'ancêtre de ce site. L'objectif était pour moi d'apprendre et de mettre en pratique le développement web full stack, j'ai alors choisi la stack **MERN** (MongoDB Express React Node.js).

J'ai conçu :

- Le front-end en **React**
- Le back-end en **Node.js** avec Express
- Le stockage dans une base de données **MongoDB**

J'ai également ajouté un système d'envoi de fichier avec *multer* et un mécanisme **CRUD**. Au cours de ce projet, j'ai beaucoup appris !
