My first big web project, the predecessor of this site. The goal was to learn and practice full-stack web development, so I chose the **MERN** stack (MongoDB Express React Node.js).

I designed:

- The front-end in **React**
- The back-end in **Node.js** with Express
- The storage in a **MongoDB** database

I also added a file upload system with _multer_ and a **CRUD** mechanism. Throughout this project, I learned a lot!
